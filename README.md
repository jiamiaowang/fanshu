# 番属
一个动漫爱好者交流的平台，目前实现发起投票、发布文章、生成文章、生成投票结果等功能。

编辑文章用富文本实现，由于是用原生代码写的，编辑文章功能和显示文章部分还有很多不足

后台是采用php+mysql，目前使用本地服务器。bakcground文件中是后台文件。

登录时密码用md5+时间加密进行判断
